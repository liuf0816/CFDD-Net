# CFDD-Net
CFDD-Net for Skin Lesion Segmentation
CFDDNet.py contains codes for CFDD-Net model.
select.py is used to split the HAM10000 dataset into training, validation and testing sets. The random seed is given for reproduction.
The other *.py files are some codes for some peripheral tests.
Some Trained weights are given in the folder "weights", where the weights for CFDD-Net is t.CPFNet-ISIC2018_min_loss.pth.
